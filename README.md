# Operations Intelligence Platform (React + MUI + Tailwind)

Feature-based React scaffold for an internal operations platform covering:
- Inventory visibility
- Production scheduling
- Order/flow tracking foundations
- Plant performance reporting

## Stack
- React + TypeScript + Vite
- Redux Toolkit + React Redux
- React Router
- Axios
- MUI + Tailwind CSS

## Run
1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`

## Required Structure
The repository implements the requested feature architecture under `src/` with 3 feature modules: `auth`, `inventory`, `production`.
